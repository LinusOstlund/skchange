## Installation
```sh
pip install git+https://github.com/NorskRegnesentral/skchange
```

## Dependencies
- `pandas` >= 1.3
- `numpy` >= 1.19
- `numba` >= 0.56
- `sktime` >= 0.24

You also need Python >= 3.8.

Optional dependencies:
- Penalty tuning: `optuna` >= 3.1.1
- Plotting: `plotly` >= 5.13.0.
